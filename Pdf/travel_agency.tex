		\item S: Service

Original version (with delegation)
	$$
	\gtRec{t}{
		\gtComm{C}{A}{
			{{Request}[String]{
					\gtComm{A}{C}{
						{{Price}[double]{
								\gtComm{C}{A}{
									{{Accept}{\gtComm{A}{S}{
												{{Delegate}[?Address.!Date]{
														\gtComm{C}{S}{
															{{Address}[String]{
																	\gtComm{S}{C}{
																	{{Date}[Date]{\gtEnd}}
																}
															}}
														}
												}}
									}}}
									{{Reject}{\gtEnd}}
									{{Repeat}{\gtVar{t}}}
								}
						}}
					}
			}}
		}
	}
	$$

Without delegation (notice that we have to add a message to $\participant{S}$
in the $\label{Reject}$ branch, otherwise the branches can not be merged.
